# Emotion-Aware AI Tutor 🎓🤖

A Python-based real-time intelligent tutoring system that adapts content based on student emotions.

### 🔥 Features
- Real-time emotion detection using DeepFace
- Adaptive slide transitions based on happiness
- Automatic hint generation when confusion detected
- Quiz interaction with scoring
- Detailed analytics dashboards (PNG graphs)
- CSV-based learning performance logging

### 🧠 Tech Stack
- Python, OpenCV, DeepFace
- Matplotlib, Pandas, CSV Logging

cd emotion-tutor
.\venv\Scripts\activate
